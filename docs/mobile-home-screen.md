# 교육팀 TMS — 모바일 홈 화면 추가

팀원·팀장이 **Safari(iPhone)** 또는 **Chrome(Android)** 에서 앱처럼 바로 열 수 있도록, **조회용**과 **편집용(팀장)** 바로가기를 각각 만듭니다.

## URL (복사용)

| 용도 | URL |
|------|-----|
| **조회** (팀원·열람) | https://okestro-edu-team-tms.vercel.app/ |
| **편집** (팀장·작성) | https://okestro-edu-team-tms.vercel.app/?mode=edit |

동일 서비스: https://edu-team-tms.vercel.app/ (별칭)

---

## iPhone (Safari)

### 1) 조회용 아이콘

1. Safari에서 **조회 URL**을 연다.
2. 하단 **공유** 버튼(□↑) 탭.
3. **홈 화면에 추가** 선택.
4. 이름 예: `교육팀 장부` 또는 `TMS 조회` → **추가**.

### 2) 편집용 아이콘 (팀장만)

1. Safari에서 **편집 URL** (`?mode=edit`) 을 연다.
2. **공유 → 홈 화면에 추가**.
3. 이름 예: `TMS 편집` → **추가**.

> 조회·편집은 **URL이 다르므로** 아이콘을 **두 개** 만들어 두는 것이 좋습니다.

---

## Android (Chrome)

1. 해당 URL(조회 또는 편집)을 연다.
2. 메뉴(⋮) → **홈 화면에 추가** 또는 **앱 설치**.
3. 이름을 구분해 저장한다.

---

## 앱 안에서

사이드바 하단 **「홈 화면에 추가」** 버튼을 누르면 위 안내와 URL 복사가 나옵니다.  
**「iPhone용 안내 — 메모에 복사」** 로 iPhone 메모 앱에 붙여넣을 체크리스트를 복사할 수 있습니다.
