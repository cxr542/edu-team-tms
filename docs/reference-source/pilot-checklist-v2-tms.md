# KPI 파일럿 체크리스트 v2 (TMS SoT)

**전제:** 공식 기록 = TMS · 엑셀 = 분석 export만

## 매주

- [ ] 일지(A) 실작업·M/M 입력
- [ ] KPI 탭 01c 주간메모 (선택, 1~2줄)
- [ ] KPI2 효과 건 토글·기준시간

## 월말

- [ ] 월마감: 01c 합 → 월 확정 반영 · 가용 M/M 확인
- [ ] KPI1 **제출** (구성원 A/B/C)
- [ ] KPI2 효과 건 **제출**
- [ ] 팀장: `module=kpi-approve`에서 **승인**
- [ ] `module=kpi-report` 월간 리포트 확인

## 분기 (2Q 시범)

- [ ] KPI3 월 메모 1건 이상 (선택)
- [ ] KPI3 분기 4요소 입력 · **분기 확정**
- [ ] 분석 Excel export (`보내기` 탭)

## 성공 기준

- [ ] 3인 KPI1+2 TMS 승인 1회 이상
- [ ] 리포트에서 등급·가동률 확인
- [ ] `npm run publish:kpi` 로 스냅샷 공유
