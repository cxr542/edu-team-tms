# KPI2 생산성 (v5 / 엑셀 요약)

> 규칙: [config/kpi2_rules.yaml](../../config/kpi2_rules.yaml)

## 정의

**생산성** = 계획시간 ÷ 실작업시간 (100% = 기준). 값이 클수록 계획 대비 빠르게 완료.

## 건별

```
생산성% = 계획시간 / 실작업시간
```

## 월 집계 (승인 건)

```
월 생산성% = Σ(계획) / Σ(실작업)
```

## 분기

```
분기 생산성% = 승인 월 생산성%의 산술평균 (0% 초과 월만)
```
