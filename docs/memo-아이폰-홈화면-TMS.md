# [나중에 할 일] iPhone — 교육팀 TMS 홈 화면 추가

> iPhone·iPad는 **Safari**로만 「홈 화면에 추가」가 됩니다. (Chrome·카톡 링크 X)

## URL

- **조회 (팀원):** https://okestro-edu-team-tms.vercel.app/
- **편집 (팀장):** https://okestro-edu-team-tms.vercel.app/?mode=edit

---

## 체크리스트

### 조회용 아이콘

- [ ] iPhone **Safari**에서 조회 URL 연다
- [ ] 하단 **공유** (□↑) 탭
- [ ] **홈 화면에 추가** 선택
- [ ] 이름: `교육팀 장부` (또는 `TMS 조회`) → **추가**

### 편집용 아이콘 (팀장만, 선택)

- [ ] Safari에서 **편집 URL** (`?mode=edit`) 연다
- [ ] **공유 → 홈 화면에 추가**
- [ ] 이름: `TMS 편집` → **추가**

---

## 참고

- 조회·편집은 주소가 달라 **아이콘 두 개**를 만들 수 있음
- **Android**는 Chrome → 메뉴(⋮) → 홈 화면에 추가
- PC에서 TMS 사이드바 **「홈 화면에 추가」** 에도 같은 안내 있음
