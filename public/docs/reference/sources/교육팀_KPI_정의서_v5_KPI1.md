# KPI1 가동률 (v5 정의서 요약)

> 원본: `교육팀_KPI_정의서_최종v5.docx`  
> 운영 엑셀 매핑: [traceability.md](../traceability.md)  
> 기계 판독 규칙: [config/kpi1_rules.yaml](../../config/kpi1_rules.yaml)

## 정의

**가동률**은 구성원의 **가용 M/M** 대비 **핵심 업무에 투입된 M/M** 비율이다.

- **업무 M/M**: 강의·기획·운영 등 KPI1 대상 업무
- **생산성향상 M/M**: 업무 효율·도구 개선 등 (KPI1 분자에 포함)
- **휴일 M/M**: 휴가·공휴 등 가용에서 제외·반영하는 M/M (운영 엑셀 G열)

## 월 산식

```
총업무진행 M/M = 업무 + 생산성향상 + 휴일
가동률(%) = 총업무진행 / 가용 M/M   (가용 0이면 산출 안 함)
```

## 주간 → 월 (운영)

1. **매주**: `01c_KPI1_주간메모`에 주×구성원 1행 (기억 보조, 집계 미반영)
2. **월말**: `01_KPI1_입력` E~G에 월 확정, **P~R 주간합**과 대조
3. 상태: 작성중 → 제출 → **승인** 후 분기 집계에 반영

## 분기 산식

```
분기 가동률 = Σ(승인 월의 총업무진행) / Σ(승인 월의 가용)
```

## 분기 등급 (가동률 %)

| 등급 | 기준 |
|------|------|
| S | 100% |
| A | 99% 이상 |
| B | 96% 이상 |
| C | 90% 이상 |
| D | 90% 미만 |

## 시뮬레이터 전용

- **일별(월~금)** 입력은 교육·체험용. 운영 엑셀에는 일별 시트가 없다.
- 일별 합 → 해당 **주 1행** → 같은 달 주간합 → 월 E~G와 검증.
